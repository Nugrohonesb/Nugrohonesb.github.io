Core Stage merupakan sebuah website yang memiliki fungsi utama sebagai pengolah data dari 4 rangkaian data yaitu student(pelajar), teacher(guru), course(pelajaran), dan grade(nilai). Dalam mengolah data core Stage memiliki beberapa fitur di dalamnya seperti read, create, update dan delete.


Pada Core Stage yang dibuat oleh kelompok 1 ini ada beberapa hal yang harus diubah jika ingin di gunakan pada laptop/PC masing-masing orang yaitu :

1. FILE CONFIG.PHP yang terletak pada folder asset > inc config.php
	Pada baris ke 2, eko25nugoo harus diganti dengan password database masing masing. Jika di database anda tidak menggunakan password, maka eko25nugoo hanya dihapus saja.

2. FILE PROSES_BACKUP yang terletak pada folder page > setting > proses_backup.php
	Pada baris ke 4, eko25nugoo harus diganti dengan password database masing masing. Jika di database anda tidak menggunakan password, maka eko25nugoo hanya dihapus saja.

3. FILE PROSES_RESTORE yang terletak pada folder page > setting > proses_restore.php
	Pada baris ke 4, eko25nugoo harus diganti dengan password database masing masing. Jika di database anda tidak menggunakan password, maka eko25nugoo hanya dihapus saja.
